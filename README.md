# Flappy Bird (Java)

A simple Flappy Bird clone developed using Java Swing and AWT.
The game demonstrates core Object-Oriented Programming concepts,
GUI development, collision detection, scoring system, and file handling.

## Features
- Gravity-based bird movement
- Moving pipe obstacles
- Keyboard or mouse controls
- Score tracking and file storage
- Retry option after game over

## Technologies Used
- Java
- Java Swing & AWT
- File Handling
